# Drone & Autonomous Boat Guidance Control Simulations


Interactive simulations of **PID altitude control for a drone** and **path‑tracking guidance for an autonomous boat**, implemented in Python with Jupyter widgets. Includes wind/current disturbances and a bonus figure‑eight trajectory for a drone.

## 📂 Notebook Contents

- **Task 1** – PID altitude controller for a drone (1‑DOF) with a step wind disturbance.  
  Use sliders to tune P, I, D gains and wind force; see the response plot and live animation.
- **Task 2** – Guidance & path tracking for a boat following a sinusoidal path.  
  Tune PD gains and water current components (x, y) to reject disturbances.
- **Bonus** – Drone tracking of a figure‑eight (lemniscate) trajectory with constant wind.

All simulations run directly in the browser – no local installation required if you use Jupyter’s interactive widgets.
